# Blog-grpc-server

gRPC Services and Methods-

PostService-

1. CreatePost - Create a new blog post for an author.
2. GetPost - Fetch a post by ID.
3. ListPosts- Returns a list of published posts.
4. PublishPost - Marks a post as published.
   

UserService-

5. CreateUser	- Creates a new user with username, email, and password.


CommentService-

6. AddComment -	Adds a comment to a post.



#Build Command - ./gradlew build


#Running the Server - ./gradlew bootRun
